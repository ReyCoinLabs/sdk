# 🟣 REY Capital — Solana Ecosystem

REY Capital is a Web3 ecosystem built on Solana, combining **DeFi, RWA, and NFTs** into a fast, scalable, and transparent infrastructure.

---

## 🌐 Official Links

* Twitter/X: https://x.com/ReycoinLabs
* Website: https://rey-coin.click/

---

## 🚀 Features

* ⚡ High-speed staking on Solana
* 🪙 SPL Token integration (REY)
* 🏢 RWA tokenization ready
* 🎨 NFT-based membership (extendable)
